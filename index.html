Jeeves - Voice Activated Home Automation

The system consists of an Arduino connected to various home appliances[Illustrated in this case using LEDs]. The system has an accompanying mobile app that connects to the hardware over bluetooth. The app converts the user's voice input into text ;using the Google Voice API;and tranfers it over to the Arduino where the string is parsed(Some simple Natural Language Processing) and the appropriate action is taken i.e. the appropriate appliance is turned on/off.

Optimisation Of Digital Advertising using Heatmaps

view detail

Optimisation Of Digital Advertising in Commercial complexes(like Malls) using Heatmaps

The system consists of multiple hardware components that are responsible for detecting human presence. For the case of the project the hardware component is created using Infra Red sensors and an arduino; Each individual Hardware component relays human presence info to the software backend. The backend recieves human presence data and creates heatmaps based on temporal factors. Based on trends identified using the heatmaps advertising is customized - EG: Heatmaps show concentration of people in the Foodcourt on Wedmesday Afternoons so digital advertising targets customers with Food related advertisements
